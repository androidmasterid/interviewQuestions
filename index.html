Higher-order components

  • Higher-order component is a function that takes (wraps) component and returns a new component.
  • Higher-order components are pure functions that take components and return new components.
  • The main purpose of a higher-order component in React is to share common functionality between components without repeating code.

    • Props Proxy (ppHOC)
    • Inheritance Inversion (iiHOC)
